Inkwave Beta Web App - Package
------------------------------
Files:
- index.html
- css/styles.css
- js/app.js
- assets/logo.png
- assets/sample1.png, sample2.png, sample3.png

How to run:
1) Unzip the package locally.
2) Open index.html in Chrome/Firefox (no server required).
3) Try the drawing canvas, AI Suggest (simulated), playback, and Export PNG.
4) To create mobile builds, wrap the site with Capacitor or Cordova and follow platform build steps.

This is a beta interactive prototype for Inkwave.
